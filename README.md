# vibrogen

Имитатор сигналов вибродиагностики на Qt6/C++.

## Что реализовано

- Генерация и вывод в стандартное аудиоустройство трёх типов сигналов:
  - гармонический сигнал;
  - белый шум;
  - сигнал типового дефекта (модулированный по скорости движения).
- Сценарии изменения скорости подвижного состава:
  - постоянная скорость;
  - разгон;
  - торможение;
  - циклический профиль (остановка → разгон → равномерное движение → торможение).
- Интерфейс приложения:
  - строка меню;
  - панель управления воспроизведением (кнопка Старт/Стоп);
  - строка состояния;
  - таблица текущего набора сигналов с добавлением/удалением;
  - панель из трёх вкладок (осциллограмма, спектр, waterfall);
  - диалоги параметров сценария и характеристик узлов ходовой части.
- Поддержка сохранения/открытия проекта генерации сигналов в JSON.
- Базовая конфигурация узлов: ЭС2Г «Ласточка» (максимальная скорость 150 км/ч).

## Сборка

```bash
cmake -S . -B build
cmake --build build
```

## Платформы

Проект целится в Windows, Linux и Android (через Qt6).
